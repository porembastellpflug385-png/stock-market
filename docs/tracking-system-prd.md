# Tracking System PRD

## 1. 产品名称

Market Analyzer Pro: Watchlist + Paper Portfolio Validation System

## 2. 产品目标

把当前的“单次分析工具”升级成一个“持续跟踪 + 模拟执行 + AI 验证”的研究平台。

核心要解决 4 个问题：

1. 哪些标的值得持续跟踪
2. 不同风格策略在同一组标的上的执行结果如何
3. AI 每日分析结论是否真的经得起市场验证
4. 哪类市场、哪类标的、哪类策略更适合使用 AI 结论

## 3. 目标用户

1. 个人投资者
2. 半专业交易者
3. 研究型用户
4. 希望用 AI 做投研辅助，但不想只看“单次报告”的用户

## 4. 核心设计原则

1. 先验证，再扩展
2. 先做“风格代理人”，不直接做真实机构持仓复制
3. 所有 AI 结论都必须可回看、可评分、可归因
4. 模拟盘不是展示收益，而是用来验证方法

## 5. 核心模块

### 5.1 关注池

功能：

1. 管理关注资产列表
2. 按市场分组：A股、港股、美股、ETF、加密
3. 自定义标签：核心观察、事件驱动、高波动、长期跟踪
4. 记录加入原因、风险备注、重点观察项

输出：

1. 今日重点观察列表
2. 风险排序列表
3. 长期跟踪列表

### 5.2 AI 分析中心

功能：

1. 按日生成单标的分析报告
2. 固定报告结构：结论、指标拆解、风险、建议、置信度
3. 保存历史报告版本
4. 支持按时间轴回看某只标的的观点变化

输出：

1. 标的日分析
2. 组合视角分析
3. AI 观点变动记录

### 5.3 模拟盘中心

功能：

1. 支持多个独立策略组合
2. 每个组合有独立现金、持仓、交易记录、净值曲线
3. 每次交易都记录触发原因
4. 支持手动与规则驱动的模拟执行

输出：

1. 组合净值曲线
2. 持仓快照
3. 交易日志
4. 风险敞口

### 5.4 策略代理人中心

第一阶段建议的代理人：

1. 全球价值基金风格
2. 全球成长基金风格
3. 宏观对冲风格
4. 幻方量化风格
5. ETF 轮动风格

每个策略代理人要定义：

1. 标的池范围
2. 入场条件
3. 加仓条件
4. 减仓条件
5. 止损规则
6. 仓位限制
7. 调仓频率

### 5.5 验证中心

功能：

1. 验证 AI 方向判断
2. 验证支撑阻力判断
3. 验证交易建议是否适合执行
4. 验证高置信结论是否更有效
5. 验证 AI 结论与不同策略风格的适配度

输出：

1. 命中率面板
2. 置信度有效性
3. 分市场/分资产类别/分策略验证结果
4. 失效案例复盘

### 5.6 报告中心

功能：

1. 日报
2. 周报
3. 月报
4. AI 验证报告
5. 策略归因报告

输出：

1. PDF / HTML / Markdown
2. 时间归档
3. 可按组合和标的过滤

## 6. 核心流程

### 6.1 日常流程

1. 系统抓取关注池最新行情
2. 生成当日 AI 分析报告
3. 各模拟策略根据规则产生动作
4. 记录持仓变化和交易日志
5. 生成日终日报
6. 更新验证结果

### 6.2 周期流程

周报：

1. 汇总本周收益
2. 汇总回撤与胜率
3. 汇总 AI 命中率
4. 输出最有效/最失效的结论类型

月报：

1. 汇总月度表现
2. 做组合归因
3. 做 AI 结论验证分层
4. 给出下月策略调整建议

## 7. AI 验证框架

### 7.1 基础验证指标

1. 1日方向命中率
2. 5日方向命中率
3. 20日方向命中率
4. 支撑位有效率
5. 阻力位有效率

### 7.2 进阶验证指标

1. 建议执行后的收益偏差
2. 高置信结论胜率
3. 不同市场环境下的命中率
4. 不同策略风格下的适配度

### 7.3 AI 结论评分字段

每份 AI 报告建议产出结构化字段：

1. 方向：看多 / 中性 / 看空
2. 置信度：0-100
3. 持有周期：短线 / 波段 / 中线
4. 建议动作：买入 / 观察 / 减仓 / 回避
5. 关键价位：支撑、阻力、止损、目标

## 8. MVP 范围

第一版只做最小闭环：

1. 关注池
2. 2 个策略代理人
3. 单标的 AI 日报
4. 模拟盘交易记录
5. 日报 / 周报
6. 方向命中率验证

建议先做的 2 个策略：

1. 成长趋势策略
2. 量化轮动策略

## 9. 非 MVP 范围

第一版先不做：

1. 真实复制 top10 基金公司持仓
2. 高频日内模拟
3. 真正的自动下单
4. 极复杂衍生品策略
5. 用户多账户权限系统

## 10. 成功指标

1. 日报稳定产出率 > 95%
2. 模拟盘交易记录完整率 > 99%
3. 关注池资产可追踪率 > 95%
4. AI 验证面板可回看覆盖率 > 95%
5. 用户能明确识别“哪些 AI 结论靠谱”

## 11. 风险与约束

1. 不同市场数据源稳定性不同
2. 真实机构策略无法完全复刻
3. AI 报告必须结构化，否则难以验证
4. 第一版应该控制复杂度，避免一开始做过多策略
